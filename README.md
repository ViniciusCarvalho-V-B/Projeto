# Sistema de Gerenciamento de Estoque

Projeto acadêmico desenvolvido com **HTML, JavaScript, Node.js e MySQL** para gerenciamento de produtos em estoque.

## 📌 Sobre o projeto

Este sistema foi criado para organizar o controle de estoque, permitindo operações de cadastro e gerenciamento de produtos por meio de interface web integrada ao back-end.

## ⚙️ Tecnologias utilizadas

- **Front-end:** HTML, CSS, JavaScript
- **Back-end:** Node.js
- **Banco de dados:** MySQL

## 🧩 Funcionalidades

- Cadastro de produtos no estoque
- Visualização de itens cadastrados
- Organização das informações em interface web
- Integração com banco de dados MySQL
